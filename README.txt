To play the game:

Open the Warp Dash folder.

Click "Warp_Dash.exe" and this will begin the game.

To play, select "Singpleplayer" and select a character.

The goal is to get as far as possible and score as many points as possible.


To view the script's source code:

Open the Source Code folder.

Open any script to view it.

(Unity requires large amounts of files associated with each project. The code listed here is the scripts that we created within Unity.)
